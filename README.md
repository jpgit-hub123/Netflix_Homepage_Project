# NetFlix Homepage Project
    A modern Netflix-inspired homepage built with HTML5 and CSS3 to practice front-end web development.

# Live Demo
     https://jpgit-hub123.github.io/Netflix_Homepage_Project/

📌 Project Overview 
    This project is a Netflix-inspired homepage built using HTML5 and CSS3. 
    The purpose of this project was to improve my front-end web development skills by recreating a modern streaming platform interface.
    
    The design includes a navigation sidebar, banner, user profile section, cards, continue watching section, and a right sidebar with users.

    * Note: This project is created for educational purposes only and is not affiliated with Netflix.
    
🚀 Features 
    🎨 Modern Netflix-inspired UI 
    📌 Sidebar navigation 
    🔍 Search bar 
    🎬 Hero banner section 
    👤 User profile area 
    🎉 Party cards 
    📺 Continue Watching cards 
    ➕ Right sidebar with friends list 
    ✨ Hover animations 
    📱 Clean and organized layout

🛠️ Built With 
    - HTML5 
    - CSS3 
    - Flexbox 
    - CSS Positioning 
    - CSS Transitions 
    - CSS Transform 
    - Box Shadow 
    - Border Radius and more!!!
